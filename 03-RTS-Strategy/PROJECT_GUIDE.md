# 🏰 RTS-Strategy - 项目导游图

## 游戏效果

即时战略游戏，控制单位采集资源、建造建筑、训练士兵、击败敌人。

**核心玩法**：
- 框选单位移动
- 采集资源（木材、矿石、金币）
- 建造生产建筑
- 训练战斗单位
- 科技树升级

## 🏗️ 核心系统

### 1. 行为树系统
- Selector：选择节点
- Sequence：顺序节点
- Task：具体行为（移动、攻击、采集）

### 2. 单位系统
- Unit：基础单位类
- Character：角色单位
- Building：建筑单位

### 3. 资源系统
- 木材、矿石、金币
- 资源采集和消耗

### 4. 科技树
- 解锁新单位
- 升级属性
