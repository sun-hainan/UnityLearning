# Contributing to UnityLearning

感谢你对 UnityLearning 项目的关注！

---

## 如何贡献

### 报告问题

如果你发现错误或有问题：

1. 检查是否已有类似 Issue
2. 创建新 Issue，包含：
   - 问题描述
   - 复现步骤
   - 预期 vs 实际结果
   - 环境信息（Unity版本、操作系统）

### 提交改进

1. Fork 本仓库
2. 创建分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m 'Add some feature'`
4. 推送分支：`git push origin feature/your-feature`
5. 创建 Pull Request

---

## 内容规范

### 教程文档

- 使用清晰的标题层级
- 代码块标注语言
- 关键步骤加粗
- 添加截图说明

### 代码示例

- 包含必要的注释
- 使用英文命名（类、方法、变量）
- 适配 Unity 2021.3+ 和团结引擎 1.6+

---

## 行为准则

- 尊重他人
- 接受建设性批评
- 关注项目最佳利益
