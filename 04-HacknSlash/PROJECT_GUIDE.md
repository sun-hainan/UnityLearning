# ⚔️ HacknSlash - 项目导游图

## 游戏效果

动作 RPG 游戏，控制角色战斗、装备、技能、探索。

**核心玩法**：
- WASD 移动
- 鼠标点击攻击
- 技能释放
- 装备系统
- 背包管理

---

## 📁 项目结构

```
04-HacknSlash/
├── source/
│   ├── Assets/
│   │   ├── Scripts/
│   │   │   ├── Player/          # 玩家控制
│   │   │   ├── Enemy/           # 敌人AI
│   │   │   ├── Inventory/       # 背包系统
│   │   │   └── Skills/          # 技能系统
│   │   └── ...
└── demo/
```

---

## 🏗️ 核心系统

### 1. 战斗系统
- 普通攻击连招
- 技能释放
- 伤害计算

### 2. 装备系统
- 武器、防具
- 属性加成
- 外观变化

### 3. 背包系统
- 物品存储
- 拖拽操作
- 使用/装备

### 4. 敌人 AI
- 状态机
- 巡逻、追击、攻击
- 掉落物品
