# 📝 示例代码库

这里包含一系列从零开始的最小可运行示例，每个示例都是一个完整的 Unity 项目。

## 示例列表

### 01-HelloCube
**功能**：创建一个旋转的 Cube  
**学习点**：Scene 视图、GameObject、脚本基础  
**预计时间**：15 分钟

### 02-PlayerMovement
**功能**：WASD 控制角色移动  
**学习点**：Input 系统、Transform、Time.deltaTime  
**预计时间**：30 分钟

### 03-PhysicsJump
**功能**：空格跳跃，物理系统  
**学习点**：Rigidbody、ForceMode、碰撞检测  
**预计时间**：45 分钟

### 04-CollectCoins
**功能**：收集金币，计分  
**学习点**：触发器、UI 显示、销毁对象  
**预计时间**：1 小时

### 05-2DPlatformer
**功能**：完整 2D 平台跳跃  
**学习点**：2D 物理、动画、地面检测  
**预计时间**：2 小时

## 使用方法

1. 进入对应示例文件夹
2. 复制项目到自己的工作目录
3. 在 Unity 中打开
4. 对照 README 学习

## 学习建议

- 先运行看效果
- 再阅读代码理解原理
- 最后自己不看代码实现一遍
